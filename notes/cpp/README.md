# C++ 高级程序设计

介绍 C++ 的一些神秘特性，课程本身也和 C++ 一样神秘。

## 课程总评分布

| 类别   | 占比  | 备注            |
| ---- | --- | ------------- |
| 平时作业 | 20% | 类似于 CPL，应全部拿满 |
| 机考   | 30% |               |
| 期末考试 | 50% | 闭卷            |

## 机考

较为简单，结合考试前上到的内容出题。

计分规则：

* 四次机考里面选三次最好的
* 考后 24H 可补交，差额取 60% 加分
* 由于评测机性能较差，考后可选择考试时提交的重测，重测若分数增加，会全额加分

## 期末考试

每年都是新题，往年卷仅供参考，尽量结合课上内容填满即可。

题型：

* 简答题：考察语言机制、设计理念等
* 看代码写结果：判断代码能否编译，若能编译写出输出，若不能编译说明原因（代码里有很多坑，千万当心）
* 手写代码题：整套试卷里最正常的题型
