# 软件测试

**软件工程方法方向的必修课**。本研共修，修的人特别多。主要讲了软件测试的很多基本概念，实验是上手做自动化测试、变异测试等，还有嵌入式环节（上板子测）。7 次 lab + 期末考试 + 慕课。

有期末考，课堂和实验都不点名。期末大量慕课原题很善良，有重点。陈老师和房老师讲得也很生动，小白听完也可以去面试唬人（）。

## 课程总评分布

原本在课堂上说的是论文综述（20%），代码大作业（30%），期末考试（40%），论文综述课堂报告（5%），课堂小测（5%），实际执行起来上面的只剩下了期末考试……

而且今年课改添加了实验课，所以给分占比尚不明确，给分好坏中等。

| 类别   | 占比   | 备注                   |
| ---- | ---- | -------------------- |
| 慕课   | 0%？  | 个人作业，最后也许是不占分（可能会变动） |
| 实验课  | 60%？ | 个人作业，但是要组小组。。？       |
| 期末作业 | 40%？ | 慕课有很多原题              |

## 实验课

* 单双周一次机房上机，不点名，但是会讲解一下注意事项，后面用板子的几次必须去才能做完。
* 建议找靠谱的小组队友合作完成。大大减少工作量。
* 经常需要装配环境，有一定操作度

## 期末考试

* 选择 + 简答 + 计算 / 问答
* 最后一节课会画重点
* 慕课有大量原题，建议考前加急
* 考试难度合理适中，很选修的选修课
