# 机器学习（深度学习）

**数据科学方向的必修课**。这门课从传统机器学习讲到了当今 LLM 的发展历程。

## 课程概览

以 2025 Fall 的安排，包含以下主题：

* **统计机器学习四大任务串讲**：即回归、分类、聚类和降维。讲的知识点基本涵盖统计机器学习基础。
* **深度学习经典结构**，包括：
  * 整个深度学习领域最基本的前馈神经网络（FNN）。
  * 几年前在图像处理领域最重要的、最基本的卷积神经网络（CNN）。
  * 以前在自然语言处理领域最早期的循环神经网络（RNN）。
* **生成式人工智能**，包括：
  * 第一个在图像生成领域取得巨大突破的网络结构——生成式对抗网络（GAN）及相关改进和发展。
  * 游戏 AI、围棋 AI 的基础，如今大模型应用的重要理论基础——强化学习（RL）。
  * 上帝的馈赠——Transformer：Attention is all you need。
* **现代大模型的理论基础、模型结构和常见应用**，包括：
  * BERT、T5、GPT 的预训练。
  * 模型的分布式训练。
  * 推理产出 token 的采样策略。
  * 微调、对齐和 RL（对齐和 RL 忘了有没有讲了）。
  * RAG、Agent 等上层应用。

目前课程还正在积极探索如何教好机器学习。
