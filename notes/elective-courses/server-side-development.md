# 服务端开发

**互联网+软件技术方向的必修课**。主要内容是讲解SpringBoot的特性和细节。企业的老师会轮流讲解Java八股，非常细节。

课堂形式：老师在群里发一个示例project，然后上课对着这个project用IDEA Debug，边De边讲解今天的Java八股，讲得很深入很细节，对于志在Java后端且想要深入研究的同学很有帮助。

没有期末考，不点名，只需要提交期中和期末作业。及格比较轻松

助教和企业老师对checklist的执行比较严格，没遵循会真的不给分（所以建议算好分数）

## 期中作业
* 实现一个创意游戏功能
* 是个人完成，课上展示有奖励分
* 25 fall是使用Thymeleaf + SpringBoot完成一个网页小游戏，有checklist和得分点

## 期末作业
* 使用Vue + SpringBoot + 数据库实现管理系统
* 有详细checklist，会严格遵守打分
* 答辩在假期线上，会追问细节和拷打，不过打分按照客观得分点
